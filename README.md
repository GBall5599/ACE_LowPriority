# ACE_LowPriority

当前发布版本：`1.3`

这是一个可直接运行的 Windows 工具，用来调整 AntiCheatExpert 相关进程的运行属性，尽量降低其对系统性能的影响。

## 文件说明

- `ACE_LowPriority.exe`：主程序，双击即可运行
- `应用说明.txt`：使用说明文档

## 软件功能

程序会对以下目标进程进行处理：

- `C:\Program Files\AntiCheatExpert\SGuard\x64\SGuard64.exe`
- `C:\Program Files\AntiCheatExpert\SGuard\x64\SGuardSvc64.exe`

处理内容：

- 将进程优先级设置为 **低**（Windows 对应 `Idle`）
- 将 CPU 亲和性限制到 **最后一个逻辑处理器**

## 1.3 版本特性

- 启动时自动请求管理员权限
- 单实例运行，重复打开时会自动唤醒已运行窗口
- 支持最小化到托盘后台运行
- 支持托盘菜单 `显示主界面 / 立即执行一次 / 退出程序`
- 支持标题栏内置 `设置 / 最小化 / 关闭` 按钮
- 支持设置 `开机自启`
- 支持设置 `自动执行`，默认每 5 秒轮询一次目标进程
- 关闭窗口时会弹出“隐藏到托盘 / 关闭程序”选择弹窗
- 执行成功或失败时会发送 Windows 提示消息

## 使用方法

1. 双击运行 `ACE_LowPriority.exe`
2. 程序启动时会自动请求管理员权限
3. 如需手动执行，进入界面后点击 `启动`
4. 如已开启“自动执行”，程序会在后台轮询目标进程并自动处理
5. 处理完成后：
   - 成功：显示成功界面，并发送一条 Windows 提示消息
   - 失败：显示错误详情，并发送一条 Windows 提示消息
6. 点击右上角 `设置` 可开启 `开机自启` 和 `自动执行`

## 使用前提

请确认以下条件：

- 操作系统为 Windows
- 允许程序获取管理员权限
- 目标进程已经在运行中，或者你已开启自动执行等待其启动

## 注意事项

- 本程序不会启动目标进程，只会处理已经运行中的进程
- 如果目标进程未运行且未开启自动执行，会显示失败界面
- 程序当前只会发送两类 Windows 提示消息：执行成功、执行失败
- 如果程序已经在运行，再次双击不会打开第二个实例
- 如果你需要源码、UI 设计稿和构建脚本，请切换到 `source` 分支查看

## 分支说明

- `main`：仅保留最终可执行文件和使用说明
- `source`：保留源码、UI 设计稿、构建脚本和开发文件
